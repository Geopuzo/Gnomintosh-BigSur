# Gnomintosh-BigSur
Gnomintosh BigSur is a customization suite that transforms the Linux GNOME desktop interface into a replica of macOS Big Sur.

## Features
* **Top Bar & Dock:** Features a translucent top menu bar and a floating bottom application dock.
* **Icon Pack:** Implements Apple's signature squircle (rounded square) application icons.
* **Window Styling:** Integrates rounded window corners and the classic traffic-light control buttons.
* **Visual Effects:** Applies system-wide blur effects, matching light/dark modes, and macOS fonts.
* **Extensions:** Uses GNOME extensions and pre-configured settings to replicate macOS animations and layouts.

## How to install?
Install the extension from the "Extension" folder, download it. Then download the config from the "dconf" folder. and place it in your home directory. Enter the command: dconf load /org/gnome/shell/extensions/ < extensions.dconf then download the wallpaper Then open the "logo menu" extension settings and download the logo of your choice from the "logo" folder and place it in the logos then download the icons and theme below in "Credits".
